# GitHub Pages

GitHub puede alojar nuestro repositorio y mostrar páginas estáticas en un subdominio

Si en nuestro repositorio, añadimos ficheros `HTML+CSS+JS` podemos ver el resultado en:

> nombreDeUsuario.github.io/miRepositorio

El fichero que se muestra es, por defecto, un `index.html`

Si realizas una página web (index.html) que contenga algo sencillo, súbelo a un repositorio tuyo en github

Sigue estos [pasos](https://docs.github.com/es/pages/getting-started-with-github-pages/configuring-a-publishing-source-for-your-github-pages-site) para habilitar las *pages* de GitHub

Comprueba el resultado en tu subdominio

Si quieres ver el ejemplo de este repositorio, sigue el enlace:

https://danielcastelao.github.io/github-pages
